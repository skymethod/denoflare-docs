# denoflare r2
Manage Cloudflare R2 storage using the S3 compatibility API

## Built using our R2 TypeScript API
Each subcommand makes R2 calls via our [R2 TypeScript API](https://github.com/skymethod/denoflare/tree/v0.5.12/common/r2), available as a reusable module or individual functions for your own projects.
The API is lightweight (no deps on aws-sdk or aws4fetch, only on [fast-xml-parser](https://github.com/NaturalIntelligence/fast-xml-parser) and web standards). It can even be used within workers themselves.

<Aside>

For non-Deno or non-TypeScript environments, you can create a single JS file (ESM module) with `deno bundle https://raw.githubusercontent.com/skymethod/denoflare/v0.5.12/common/r2/r2.ts`.

Or if you don't have Deno installed, using our on-demand [deno bundle as a service](https://bundle.deno.dev): `https://bundle.deno.dev/https://raw.githubusercontent.com/skymethod/denoflare/v0.5.12/common/r2/r2.ts`
</Aside>
